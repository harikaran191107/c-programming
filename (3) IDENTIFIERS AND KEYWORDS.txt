IDENTIFIERS AND KEYWORDS

Identifiers:

Identifiers are the names given to variables, functions, arrays, structures, etc. They are created by programmers to identify different program elements.

examples of Identifiers:

#include<stdio.h>
int main(){
    int marks;
    float average;
    char grade;
    scanf("");
    printf("");
    return 0;
   }

In this, 
marks
average
grade are the identifiers


Keywords:

Keywords are reserved words that already have a special meaning in the C language.You cannot use them as variable names, function names, or identifiers.

keywords in c are,
 auto      break     case      char   
 const     continue  default   do     
 double    else      enum      extern 
 float     for       goto      if     
 int       long      register  return 
 short     signed    sizeof    static 
 struct    switch    typedef   union  
 unsigned  void      volatile  while
